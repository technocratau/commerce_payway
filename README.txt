INTRODUCTION
------------
This project integrates Payway Trusted Frame online payments into
the Drupal Commerce payment and checkout systems.
https://www.payway.com.au/docs/rest.html
* For a full description of the module, visit the project page:
  https://www.drupal.org/project/commerce_payway_frame
* To submit bug reports and feature suggestions, or to track changes:
  https://www.drupal.org/project/issues/commerce_payway_frame


REQUIREMENTS
------------
This module requires the following:
* Submodules of Drupal Commerce package (https://drupal.org/project/commerce)
  - Commerce core,
  - Commerce Payment (and its dependencies);
* API Key (https://www.payway.com.au/core/DeveloperTestFacilityView)

TROUBLESHOOTING
---------------
* To test with different credit card, please go to
https://www.payway.com.au/docs/rest.html#test-card-numbers
